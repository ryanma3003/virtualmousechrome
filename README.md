## Cara Gunainnya
1. Buka halaman Extensions dengan memasukkan `chrome://extensions` di tab baru.
2. Aktifkan **Mode Developer** dengan mengklik tombol di samping Mode developer.
3. Klik tombol **Load unpacked** dan pilih direktori ekstensi.

Source untuk lebih jelasnya : https://developer.chrome.com/docs/extensions/get-started/tutorial/hello-world?hl=id#load-unpacked
